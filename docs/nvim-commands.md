# Nvim Commands

## Общие

| Команда | Действие |
|---------|----------|
| `:w` | Сохранить файл |
| `:q` | Закрыть окно / выйти |
| `:wq` | Сохранить и выйти |
| `:qa` | Закрыть все окна и выйти |
| `:checkhealth` | Диагностика состояния Neovim и плагинов |

## Плагины (Packer)

| Команда | Действие |
|---------|----------|
| `:PackerSync` | Установить / обновить все плагины |
| `:PackerUpdate` | Обновить плагины без удаления |
| `:PackerClean` | Удалить неиспользуемые плагины |
| `:PackerStatus` | Показать список установленных плагинов |

## LSP и Mason

| Команда | Действие |
|---------|----------|
| `:LspInfo` | Информация об активных LSP-серверах |
| `:Mason` | Открыть менеджер LSP-серверов, линтеров, форматтеров |
| `:MasonInstall <name>` | Установить сервер/линтер через Mason |
| `:NullLsInfo` | Информация об активных источниках null-ls |

## Treesitter

| Команда | Действие |
|---------|----------|
| `:TSUpdate` | Обновить все установленные парсеры |
| `:TSInstall <lang>` | Установить парсер для языка |

## AI

| Команда | Действие |
|---------|----------|
| `:Codeium Auth` | Авторизация в Codeium |
| `:Codeium Chat` | Открыть чат Codeium |

## Поиск (Telescope)

| Команда | Действие |
|---------|----------|
| `:Telescope find_files` | Поиск файлов по имени |
| `:Telescope live_grep` | Поиск текста по содержимому файлов |

## Диагностика

| Команда | Действие |
|---------|----------|
| `:TroubleToggle` | Открыть/закрыть панель диагностики |

## Превью

| Команда | Действие |
|---------|----------|
| `:MarkdownPreview` | Запустить превью Markdown в браузере |
| `:MarkdownPreviewStop` | Остановить превью |

---

Источники: `nvim/lua/plugins/init.lua`, `:help`
