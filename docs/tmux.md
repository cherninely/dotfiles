# Tmux

Prefix: `Ctrl+A`

## Навигация между панелями

| Клавиша | Действие |
|---------|----------|
| `prefix h` | Панель слева |
| `prefix j` | Панель снизу |
| `prefix k` | Панель сверху |
| `prefix l` | Панель справа |

## Изменение размера панелей

| Клавиша | Действие |
|---------|----------|
| `prefix H` | Уменьшить ширину (на 5) |
| `prefix J` | Уменьшить высоту (на 5) |
| `prefix K` | Увеличить высоту (на 5) |
| `prefix L` | Увеличить ширину (на 5) |

## Copy-mode (vi-style)

| Клавиша | Действие |
|---------|----------|
| `v` | Начать выделение |
| `y` | Копировать в буфер обмена (pbcopy) |
| `prefix P` | Вставить из буфера |
| Перетаскивание мышью | Копирование выделенного (pbcopy) |

## Мышь

| Клавиша | Действие |
|---------|----------|
| `prefix m` | Включить мышь |
| `prefix M` | Выключить мышь |

## Максимизация панели

| Клавиша | Действие |
|---------|----------|
| `prefix Up` | Развернуть панель (перенести во временное окно) |
| `prefix Down` | Свернуть панель (вернуть обратно) |

## Прочее

| Клавиша / Настройка | Описание |
|----------------------|----------|
| `prefix r` | Перезагрузить конфиг |
| `mode-keys vi` | Vi-режим навигации |
| `base-index 1` | Нумерация окон с 1 |
| `history-limit 10000` | Лимит истории скролла |
| `escape-time 0` | Без задержки после Esc (для vim) |

## Плагины

| Плагин | Описание |
|--------|----------|
| `tpm` | Менеджер плагинов |
| `tmux-onedark-theme` | Тема One Dark |
| `tmux-yank` | Кросс-платформенный буфер обмена |
| `tmux-resurrect` | Сохранение/восстановление сессий |
| `tmux-continuum` | Автоматическое сохранение/восстановление |
