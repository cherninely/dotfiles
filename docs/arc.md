# Arc VCS

Aliases для работы с Arc (Arcadia VCS).

## Aliases (`.aliases`)

| Alias | Команда | Описание |
|-------|---------|----------|
| `aa` | `arc add` | Добавить в индекс |
| `ab` | `arc branch -a` | Все ветки |
| `ac` | `arc commit -am` | Коммит всех изменений |
| `aca` | `arc commit --amend` | Amend коммит |
| `ad` | `arc diff` | Diff |
| `adc` | `arc diff --cached` | Diff cached |
| `ads` | `arc diff --staged` | Diff staged |
| `af` | `arc fetch` | Fetch |
| `al` | `arc log --oneline` | Лог (однострочный) |
| `ap` | `arc push` | Push |
| `as` | `arc st` | Статус |
| `at` | `arc tag` | Теги |
| `ast` | `arc status -sb` | Краткий статус |
| `art` | `arc rebase arcadia/trunk` | Rebase на trunk |
| `apf` | `arc push --force` | Force push |
