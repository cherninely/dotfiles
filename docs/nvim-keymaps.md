# Nvim Keymaps

Leader: `,`

## Навигация и окна

| Клавиша | Действие |
|---------|----------|
| `C-h/j/k/l` | Перемещение между окнами |
| `C-D` | Скролл вниз + центрирование |
| `C-U` | Скролл вверх + центрирование |
| `n` / `N` | Следующий/предыдущий результат поиска + центрирование |

## Буферы / табы (BufferLine)

| Клавиша | Действие |
|---------|----------|
| `S-h` / `S-l` | Предыдущий / следующий буфер |
| `gb` | Выбрать буфер (pick) |
| `]b` / `[b` | Переместить буфер вправо / влево |
| `<leader>x` | Закрыть буфер |
| `gs` | Сортировка буферов по директории |

## Файлы и поиск

| Клавиша | Действие |
|---------|----------|
| `<leader>p` | Telescope: поиск файлов |
| `<leader>f` | Telescope: live grep |
| `<leader>F` | Telescope: grep слова под курсором |
| `<leader>d` | NvimTree: toggle |
| `<leader>t` | TODO-комментарии (TodoTelescope) |

### NvimTree (внутри дерева файлов)

Стандартные кеймапы (конфиг: `setup {}`).

| Клавиша | Действие |
|---------|----------|
| `Enter` / `o` | Открыть файл / развернуть папку |
| `<Tab>` | Превью файла (без перехода) |
| `C-v` | Открыть в вертикальном сплите |
| `C-x` | Открыть в горизонтальном сплите |
| `C-t` | Открыть в новом табе |
| `a` | Создать файл/директорию (с `/` на конце — папка) |
| `d` | Удалить |
| `r` | Переименовать |
| `x` | Вырезать |
| `c` | Копировать |
| `p` | Вставить |
| `y` | Копировать имя файла |
| `Y` | Копировать относительный путь |
| `gy` | Копировать абсолютный путь |
| `R` | Обновить дерево |
| `H` | Показать/скрыть скрытые файлы |
| `I` | Показать/скрыть игнорируемые файлы |
| `q` | Закрыть NvimTree |
| `g?` | Показать справку по кеймапам |

## Редактирование

| Клавиша | Действие |
|---------|----------|
| `<leader>s` | Сохранить файл (normal и insert) |
| `<leader>r` | Заменить слово под курсором во всём файле |
| `<leader>y` | Yank в системный буфер обмена |
| `<leader>Y` | Yank строки в системный буфер обмена |
| `x` / `X` | Удалить символ без копирования |
| `p` (visual) | Вставить без копирования заменённого текста |
| `Q` | Отключена |
| `Enter` | Убрать подсветку поиска |
| Стрелки | Отключены (использовать `hjkl`) |

## LSP

Доступны только в буферах с активным LSP-сервером.

### Навигация

| Клавиша | Действие |
|---------|----------|
| `gd` | Go to definition |
| `gD` | Go to declaration |
| `gI` | Go to implementation |
| `gr` | References |
| `gl` | Показать диагностику (float) |

### Действия

| Клавиша | Действие |
|---------|----------|
| `<leader>ll` | Hover (документация) |
| `<leader>la` | Code action |
| `<leader>lr` | Rename |
| `<leader>lf` | Format (async) |
| `<leader>ls` | Signature help |
| `<leader>le` | Диагностика (float) |
| `<leader>lj` | Следующая диагностика |
| `<leader>lk` | Предыдущая диагностика |
| `<leader>lq` | Диагностика в loclist |
| `<leader>li` | LspInfo |
| `<leader>lI` | LspInstallInfo |
| `<leader>lt` | Trouble toggle |

## Автодополнение (cmp)

| Клавиша | Действие |
|---------|----------|
| `C-;` | Вызвать меню автодополнения |
| `Tab` | Подтвердить выбор |
| `C-s` | Отменить автодополнение |
| `C-j` | Следующий вариант / expand snippet |
| `C-k` | Предыдущий вариант / jump назад в snippet |
| `Enter` | Отключён в cmp |
