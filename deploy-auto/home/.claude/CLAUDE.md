# Global Instructions

## Serena LSP и AST Index — приоритетные инструменты для работы с кодом

При работе с кодом **всегда предпочитай Serena LSP и AST Index** встроенным средствам (Grep, Glob, Read).

### Serena LSP

- **Навигация по коду**: используй `find_symbol`, `get_symbols_overview`, `find_referencing_symbols` вместо Grep/Glob для поиска классов, функций, методов и переменных.
- **Поиск связанных частей кода**: используй `find_referencing_symbols` чтобы найти все места, где используется символ — это точнее текстового поиска через Grep.
- **Понимание структуры файла**: начинай с `get_symbols_overview` перед чтением всего файла через Read.
- **Чтение кода**: используй `find_symbol` с `include_body=True` для чтения конкретных функций/методов вместо Read для всего файла.
- **Редактирование кода**: используй `replace_symbol_body`, `insert_after_symbol`, `insert_before_symbol` для точечных правок.
- **Переименование**: используй `rename_symbol` для переименования с автоматическим обновлением всех ссылок.
- **Поиск паттернов**: используй `search_for_pattern` когда не знаешь точное имя символа.

### AST Index — ориентация в проекте

Используй `/ast-index` когда нужно **найти** код, а не работать с уже найденным:

- Не знаешь, в каком файле определён класс или функция → поиск символа по имени
- Нужно понять, кто вызывает функцию или использует тип → поиск использований
- Нужно найти все реализации интерфейса → поиск реализаций
- Нужно разобраться в незнакомой части проекта → карта модулей, иерархия классов

Нашёл через AST Index — дальше работай через Serena LSP.

### Когда какой инструмент использовать

| Задача | Инструмент |
|--------|-----------|
| Навигация внутри файла, чтение/редактирование символов | **Serena LSP** |
| Найти символ, его использования, реализации, структуру проекта | **AST Index** |
| Файлы конфигурации (JSON, YAML, TOML, .env) | Встроенные (Read, Edit) |
| Markdown, README, документация | Встроенные (Read, Edit) |
| Простые однострочные правки внутри тела функции | Edit |
| Проект не подключен к Serena/LSP недоступен | Встроенные (Grep, Glob, Read) |
